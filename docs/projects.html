Project 3: Microcomputers (Spring 2024)

In this project, I utilized the PIC24 microcontroller with MPLAB IDE to design an embedded system that controls on LED control and interacts with a switch input functionality. The system involves programming the microcontroller to control LED based on user input through a tactile switch, utilizing low-level assembly language for precise hardware manipulation.

This project allowed me to gain hands-on experience in configuring microcontroller I/O pins, implementing pull-up resistors, and managing input/output interactions, enhancing my skills in embedded systems programmming and hardware interfacing.

Project 4: Software Design and Engineering (Fall 2024)

As part of my coursework, I worked on developing and enhancing the Chocoholics Anonymous (ChocAn) system, a software solution for managing membership and service records for a fictional support group. The system's primary objective was to automate the tracking and reporting of participant activities, enabling efficient management of resources and operations.

I focused on implementing and optimizing key functionalities of the system, including:

  • Database Management: Designed and implemented SQl-based system for managing member information, service records, and group statistics. I ensured data integrity, performed normalization to reduce redundancy, and optimized queries for faster retrieval.
  • User Interface Design: Developed a user-friendly interface for members and administrators, allowing for easy navigation, real-time updates, and interaction with the system. This involved designing interactive screens for members to log activities and view reports.
  • Performance Optimization Focused on optimizing the performance of critical database operations, such as searching and sorting large datasets. I implemented efficient algorithms to improve the system's response time and ensure scalability as the system grows.
